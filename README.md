📸 PicAlbum - Android Photo Gallery App

A modern Android photo gallery app built using Java and Android SDK.  
It allows users to browse, view, and manage images from device storage with a smooth and responsive UI.

---

🚀 Features

- 📂 Load images from device using MediaStore
- 🖼️ Grid-based gallery view
- 🔍 Fullscreen image viewer
- 🔄 Swipe between images (ViewPager2)
- 🔎 Pinch-to-zoom support (ZoomageView)
- ℹ️ View image metadata (name, size, date)
- 📤 Share images via Android share sheet
- 🗑️ Delete images with system confirmation
- 🔙 Custom back navigation

---

🛠️ Tech Stack

- **Language:** Java  
- **UI:** XML + ConstraintLayout  
- **Image Loading:** Glide  
- **Zoom:** ZoomageView  
- **Architecture:** RecyclerView + ViewPager2  
- **Storage:** MediaStore API
